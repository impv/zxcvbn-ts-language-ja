# Development setup

## Requirements

- Node.js (I'd recommend to use version management tools, such as [nvm](https://github.com/nvm-sh/nvm))
- [pnpm](https://pnpm.io/) (`npm i -g pnpm`)

## Install development dependencies

```sh
$ pnpm i
```

# Commands

## Transpile TypeScript files

```sh
$ pnpm build

# watch .ts files
$ pnpm build -- --watch
```
